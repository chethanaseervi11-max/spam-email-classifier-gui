📧 Spam Email Classifier (GUI)

📌 Description

This project is a spam message classifier built using Python. It uses a machine learning model (Naive Bayes) and CountVectorizer for text processing. A simple Tkinter GUI allows users to enter a message and check whether it is spam or not.

🚀 Features

- Detects spam messages
- Simple GUI interface
- Instant prediction
- Easy to use

🛠️ Tech Stack

- Python
- Scikit-learn
- Tkinter

▶️ How to Run

1. Install required library:

pip install scikit-learn

2. Run the program:

python main.py
